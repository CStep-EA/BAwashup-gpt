## Step 3: Deploy

1. Click **Deploy** — Vercel will build from the `main` branch
2. Wait for build to complete (typically 30-60 seconds)
3. Note your deployment URL: `https://YOUR-PROJECT.vercel.app`

---

## Step 4: Update Railway CORS (CRITICAL)

Once you have the Vercel URL, update Railway's `ALLOWED_ORIGINS` env var:

1. Go to Railway Dashboard → your project → Variables
2. Set `ALLOWED_ORIGINS`:
   ```
   https://YOUR-PROJECT.vercel.app,https://your-custom-domain.com
   ```
3. Railway will auto-redeploy with the updated CORS config
4. If using a custom domain later, add it to the comma-separated list

## Step 5: Verify Deployment

### 5a. Basic page load
```bash
curl -s -o /dev/null -w "%{http_code}" https://YOUR-PROJECT.vercel.app
# Expected: 200
```

### 5b. SPA routing (deep link)
```bash
curl -s -o /dev/null -w "%{http_code}" https://YOUR-PROJECT.vercel.app/login
# Expected: 200 (served by index.html via rewrite)
```

### 5c. Security headers
```bash
curl -sI https://YOUR-PROJECT.vercel.app | grep -E "x-content-type|x-frame|referrer"
# Expected:
#   x-content-type-options: nosniff
#   x-frame-options: DENY
#   referrer-policy: strict-origin-when-cross-origin
```

```bash
curl -sI https://YOUR-PROJECT.vercel.app/assets/index-BHdfHfZI.js | grep cache-control
# Expected: cache-control: public, max-age=31536000, immutable
```

### 5e. PWA manifest
```bash
curl -s https://YOUR-PROJECT.vercel.app/manifest.webmanifest | python3 -m json.tool | head -5
# Expected: Valid JSON with name "Bower Ag CowCare Tool"
```

## Troubleshooting

### Build fails with TypeScript errors
```bash
cd frontend && npx tsc --noEmit  # Run locally first
```

### Build fails with missing dependencies
Ensure `package-lock.json` is committed (Vercel uses `npm ci`).

### API calls return CORS errors
- Verify `ALLOWED_ORIGINS` on Railway includes your Vercel URL
- Ensure no trailing slash in the origin URL
- Check browser DevTools → Network tab for preflight (OPTIONS) response

### Blank page on load
- Check browser console for JS errors
- Verify `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY` are set
- The Supabase client logs a warning if credentials are missing

### PWA not installing
- Ensure HTTPS (Vercel provides this automatically)
- Check manifest at `/manifest.webmanifest`
- Verify service worker at `/sw.js`
